Pong in C++
As part of my university course at UWE, I created a Pong clone in C++. It was created in a bespoke engine called ASGE. It is specially designed and targeted for students as it provides the minimum amount of hand-holding.

The Pong clone was an important stepping stone, as it allowed me to get familiar with some advanced programming concepts, such as pointers, memory management and classes.
