# Year One

## Pong - C++

{% include image-gallery.html folder="/assets/img/portfolio/uni/pong" %}

Pong was the first C++ game that I developed at university. It used a simple framework called [Awesome Sauce Game Engine](https://huxyuk.github.io/AwesomeSauceGE/) or ASGE for short. This framework was developed by a lecturer at my university and is specifically designed and targeted at students.

The Pong game is a simple clone of the original game which released in 1972. It includes a two player mode, where two people play against each other. A one player mode, where one person plays against an AI opponent. In both modes there are multiple difficulties, these difficulties change the speed of the ball to make it harder or easier to hit it. Below is a short video of the game played with two players in normal difficulty. As well as a download link, this only includes the built game, no source code is provided at this moment.
